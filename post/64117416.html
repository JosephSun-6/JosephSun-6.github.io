首先把空间分析下一个定义。

什么是空间分析？

空间分析是指利用拓扑关系、几何关系或地理属性等来研究事物的一种方法。严格上说空间分析指的并不是一种技术，而是一系列使用不同的分析手段并应用于各个领域的一系列方法。目前空间分析主要的应用是在地理信息系统中用来分析地理数据。

———— wiki

照我的理解，空间分析是一种解决问题的思路或方法，地理信息系统可以用于进行空间分析。

空间分析要解决什么问题？

空间分析为发现地理对象和过程的规律性或模式提供定量的描述、分析和统计依据支持。

模式包括：时空趋势、分布情况、相关性、异质性、临近性、可达性等。

了解空间分析之前，我们还需要先了解点统计学的基础知识。

概率与概率分布

概率 (Probability) - 是对一个(随机)事件发生的可能性的度量

就好比天气预报以前说明天有雨，结果没下，大家就说天气预报不准。现在聪明多了，改成了今天会下雨的概率，如果没发生，确实也有没法生的概率。

概率密度函数 (Probability density function, PDF) - 刻画连续型随机变量取不同值的相对可能性大小(relative likelihood)

概率质量函数 (Probability mass function，PMF) - 用于刻画离散型随机变量取值的相对可能性

大概率小事件 vs. 小概率大事件

大概率事件

炒股的散户大多都是赔的

小概率大事件必然发生

散户们能赚到钱(小概率事件必然发生是正确的，但是这句话的含义是小概率事件迟早发生的概率为1，并不是在某次试验中小概率事件必然发生)

分析的目的之一发现小概率大事件

随机变量的数字特征

随机变量和变量的区别是什么？先举几个随机变量的例子：骰子(离散型)、等待公交车的时间(连续型)、足彩(随机向量)。也就是说随机变量是可以用概率来描述的，而变量无法量化。

但是实际问题的概率分布通常很难确定，通过样本 (Sample) 对概率总体 (Population) 进行估计，常利用数字特征来刻画随机变量的特点。

通俗的讲，一个黑箱中多个球，我多次有放回的去摸球，摸出来的球就是样本，我用我自己的结果去估计黑箱中的原本的球构成。

样本均值 (Mean) - 变量观测值的中心趋势-总体期望μ的点估计

样本方差 (Sample variance) - 变量观测值偏离均值的程度总体方差σ^2^的无偏点估计

中位数 (Median) - 变量观测值大小中间的值

分位数 (Quantiles) - 将变量分割成连续等概率分布区间的值；四分位数 (Quartiles) 是分割为4个区间的值;

统计假设检验

统计假设检验/显著性检验（Statistical hypothesis testing)

现在谈谈怎么去进行统计研究。

核心思想为某种带有概率性质的反证法。即先认为某种假设是成立的，如果导出一个不合理的现象，则拒绝假设，称为不相容的 (incompatible)，可以接受替代假设；否则不能拒绝假设，称为相容的。

不合理的现象是指“发生了小概率事件”- 某个由样本的计算统计量位于概率分布的两端

假设检验中的假设包括两个：原假设 H~0~ 和备择假设 H~1~

选取原则：

  • 应当把如果它成立但是误判为不成立时会造成严重后果的命题作为原假设。
  • 应当把分析人员想证明它正确的命题作为备择假设，把分析人员努力证明它不正确的命题作为原假设
  • 应当把大众普遍认为它成立的命题作为原假设，因为原假设不应当轻易拒绝，除非有足够的证据表明他不正确。
参数检验和非参数检验
  • 非参数检验不意味着不使用参数（是不使用总体分布的特征参数）
  • 参数检验适用于正态总体，非参数适用于非正态总体
  • 参数检验考虑总体分布，通过估算总体参数构建检验统计量
  • 非参数检验不考虑总体分布，直接构建统计检验量
  • 参数检验利用总体信息，非参数检验利用样本信息
  • 参数检验参数固定，非参数检验参数不固定

(空间) 统计假设检验一般步骤 (Null Hypothesis Significance Testing- NHST)

  • 确定H~0~(null) 和H~1~(alternative)

  • 选择统计量与分布形式

  • 计算统计量

  • 选择显著水平α

  • 比较计算统计量和选定显著水平下的理论临界值大小，得出检验结论

    [如果计算值> (OR <, 单侧/双侧)显著水平α下的理论值，小概率事件发生，拒绝H~0~, 接受H~1~]

如果H~0~没有被拒绝，则没有显著相关性(趋势)，并发现在统计量与 …

不可以说接受H~0~！！！贝叶斯方法可以说接受！！！

Student’s t-test

  • 计算统计量t满足N-1个自由度(~独立观测个数)的Student’s t分布

  • t = mean(tmp)/sd(tmp)×sqrt(length(tmp)) # = 2.746

  • α = 0.05显著水平下8个自由度t分布的临界值：qt(0.95, df=8) # =1.859

  • 样本计算值t > 临界值，小概率事件发生，拒绝H0，接受H1

  • 临界值右侧曲线下的面积0.05，单尾检验小概率事件
    发生的区间

  • pnorm(-1.96) = 0.025

Z-Scores

  • Z-Scores (也称作：standard scores, z-values, standardized variables, normal scores)，对于一个 raw score x：

  • z满足标准正态分布N(0,1)，用于H0下检验分布可以由正态分布来近似的统计检验，称为Z-test

  • 中心极限定理(the central limit theorem)表明，对于大样本(n > 30?)来说，许多检验统计都可以由正态分布来近似 (小样本t-test更合适)

  • Z-test也特指the one-sample location test

    location参数包括mean, median, mode等

  • 一般来说H0：观测统计量(均值)=给定随机条件下的总体期望值

  • Z-test具有唯一的临界值

  • alpha=0.05
    双侧：[-1.96, +1.96]
    单侧：[-1.65, +1.65]

T检验、方差分析、非参检验、卡方检验的使用要求和适用场景

1 2

1、有关平均值参数u的假设检验

根据是否已知方差,分为两类检验：U检验和T检验。
如果已知方差，则使用U检验，如果方差未知则采取T检验。

2、有关参数方差σ2的假设检验

F检验是对两个正态分布的方差齐性检验，简单来说，就是检验两个分布的方差是否相等

3、检验两个或多个变量之间是否关联

卡方检验属于非参数检验，主要是比较两个及两个以上样本率（构成比）以及两个分类变量的关联性分析。根本思想在于比较理论频数和实际频数的吻合程度或者拟合优度问题。

Mann-Whitney U test

  • Mann-Whitney U 检验(Wilcoxon rank-sum检验)是一种非参数的检验方法，不要求满足任何分布，基于样本构造检验统计量，用来检验两个独立样本是否来自同一个总体

  • 将数值转换成次序，混合排序，对异常值不敏感，只关心相对大小排序，不关心绝对。

  • 第一步，两个样本混合排序

  • 第二步，计算每个样本秩的和，排列后的次序相加

  • 第三步，计算统计量

  • 第四步，小样本时利用Wilcoxon rank sum statistic分布精确计算p-value

正态性检验

  • 正态性检验(Normality test)是检验观测数据是否符合正态分布
  • 参数检验经常要求数据符合正态分布
  • 如果不满足正态分布，推荐使用非参数方法
Shapiro–Wilk test

  • 针对正态分布

  • H~0~: 样本来自正态总体

  • 检验统计量

The null-hypothesis of this test is that the population is normally distributed. Thus, if the p value is less than the chosen alpha level, then the null hypothesis is rejected and there is evidence that the data tested are not normally distributed. On the other hand, if the p value is greater than the chosen alpha level, then the null hypothesis (that the data came from a normally distributed population) can not be rejected (e.g., for an alpha level of .05, a data set with a p value of less than .05 rejects the null hypothesis that the data are from a normally distributed population).

Kolmogorov‐Smirnov test —— KS检验
  • 非参数检验方法, 适用于一般分布，其他分布
  • 检验统计量为样本的经验分布和参考累积分布之间的最大差
  • 当分布参数未知时, 结果不可靠, 使用 Monte‐Carlo方法

Effect Size (效应量)

  • Effect size is a statistical concept that measures the strength of the relationship between two variables on a numeric scale

  • 效应量——p值显著不显著，采用”度”描述，显著程度。

  • 各个论文中不同时间地区的结论可以相互比较的方法，计算不同效应量，然后得出对比结论

  • Meta - analysis (元分析)

    统计学上来说，元分析（meta-analysis，或译作后设分析、整合分析、综合分析、统合分析、荟萃分析）是指将多个研究结果整合在一起的统计方法。就用途而言，它是文献回顾的新方法。

    文献回顾的传统方法是叙事式的，由作者自行挑选觉得重要的前人研究，当各研究结论冲突时，由作者自行判断哪一种结论较具价值。反之，后设分析采用系统化的资料收集和数值分析，让文献回顾更具客观性与证据力。这些组合后的结果会有助于以更加全面性的证据观察来支持或拒绝某一主题研究的假设，发现该主题迄今研究的不足处，发现各种可能影响效应量的中介因素。

  • 对于T检验的效应量一般常用 Cohen’s d 效应量指标

    方法1：

    $Cohen’s d = (M2 - M_1)/SD{pooled}$

    SD~pooled~ 为联合方差 —— $ SD_{pooled} =\sqrt {\frac{SD^2_1+SD^2_2}{2}}$

    方法2：

    $d =\frac{|\overline x_1-\overline x_2|} {\sqrt{\frac{(n_1-1)s^2_1+(n2-1)s^2_2}{n_1+n_2-2} }}$
