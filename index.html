z-test

Z检验，也称“U检验”，是为了检验在H0情况下样本数据能否可以接近正态分布的一种统计测试。Z-test也特指the one-sample location test 。

$X_i$ 是样本总体数据中的第i个数据；$\mu$ 是总体均值；$\sigma$ 是总体标准差

根据中心极限定理，在大样本条件下许多测验可以被贴合为正态分布。当实际标准差未知，而样本容量较小（小于等于30）时，T-test更加适用。

Normality tests

In statistics, normality tests are used to determine if a data set is well-modeled by a normal distribution and to compute how likely it is for a random variable underlying the data set to be normally distributed.

—wiki

t-test等检验都基于数据具有某种特征，比如正态分布等，而且其有效性取决于数据的分布。因此再进行检验前，最好先验证样本数据是否符合前提假设，如果不满足，可以考虑非参数检验。今天另一个重点讲的就是Mann–Whitney U test 非参数检验。

评估样本数据正态性可以考虑以下几个方法：

  1. 大样本情况：如果样本数目大于30，可以默认为大样本情况，忽略其样本分布，采用参数检验，但最好采用目视检查或正态显著性检验等方法，查看其分布的正态性。

  2. 目视检查

    正如上一篇t-test里所写到的

    • 密度图：查看其是否符合钟型分布
    • 经验累积概率图：
    • QQ图：对照理论值和样本值分位数，查看样本与正态分布之间差别

  3. 正态分布显著性检验

    通常情况下，目视检查只能看个大概，并不可靠，可以采用显著性检验方法对样本与正态分布进行比较，以定量确定样本的偏离程度。

    通常方法包括：

    1. 图示法

      P-P图 ppplot {需要包 StatDA}

      频率直方图 hist()

      箱线图 boxplot()

      茎叶图 stem()

      茎叶图又称“枝叶图”，它的思路是将数组中的数按位数进行比较，将数的大小基本不变或变化不大的位作为一个主干（茎），将变化大的位的数作为分枝（叶），列在主干的后面，这样就可以清楚地看到每个主干后面的几个数，每个数具体是多少。

    2. 偏度、峰度检验法(冒牌 K-S 检验法)

      • S，K 检验 —— 偏度(Skewness)和峰度(Kurtosis)

        偏度 Skewness

        偏度衡量随机变量概率分布的不对称性，是相对于平均值不对称程度的度量，通过对偏度系数的测量，我们能够判定数据分布的不对称程度以及方向。

        具体来说，对于随机变量X，我们定义偏度为其的三阶标准中心距:

        其中，$\overline x$ 为样本均值，$m_3$ 为样本三阶中心矩，$m_2$ 是样本二阶中心矩

        偏度的衡量是相对于正态分布来说，正态分布的偏度为0，即若数据分布是对称的，偏度为0。若偏度大于0，则分布右偏，即分布有一条长尾在右；若偏度小于0，则分布为左偏，即分布有一条长尾在左（如下图）；同时偏度的绝对值越大，说明分布的偏移程度越严重。

        注意：数据分布的左偏或右偏，指的是数值拖尾的方向，而不是峰的位置。

        峰度 Kurtosis

        峰度，是研究数据分布陡峭或者平滑的统计量，通过对峰度系数的测量，我们能够判定数据相对于正态分布而言是更陡峭还是更平缓。比如正态分布的峰度为0，均匀分布的峰度为-1.2（平缓），指数分布的峰度6（陡峭）。

        峰度，定义为四阶中心距 $m_4$ 除以方差 $\delta$ 的平方减3。

        若峰度 $\approx$ 0 , 分布的峰态服从正态分布；若峰度 > 0,分布的峰态陡峭（高尖）；若峰度 \< 0,分布的峰态平缓（矮胖）；

    如果样本数据来自具有正态分布的总体，JB统计量近似服从自由度为2的卡方分布，即 JBasy ~$\chi^2$，因此该统计量可以用于检验数据是否服从正态分布。原假设H0是偏度为0，峰度为3(因为正态分布的偏度为0，峰度为3)，若变量不是正态分布，则JB统计量将为一个逐渐增大值。

    如果JB统计量值较大，比如为11，则可以计算出卡方值大于11的概率为0.004，这个概率过小，因此不能认为样本来自正态分布。反之，成立。

    JB统计量用来检验序列观测值是否服从正态分布，该检验的零假设为样本服从正态分布。在零假设下，JB统计量服从$\chi^2 (2)$分布，变量X服从正态分布。

非参数检验方法

在我国国标《GB/T4882-2001数据的统计处理和解释：正态性检验》中，在有方向的检验方法中，推荐了偏度和峰度检验；在无方向的检验方法中，推荐了Shapiro–Wilk检验和Epps-Pulley检验，并明确Shapiro–Wilk检验适用于8≤n≤50的样本数据，Epps-Pulley检验适用于n≥8的样本。

  • K-S检验 —— Kolmogorov-Smirnov检验

    K-S检验的思路是那样本数据和期望的理论分布进行对比，若差异不大，则可以认为服从正态分布。

    Smirnov检验则是针对两组样本数据是否服从同一分布的检验，其思想与Kolmogorov检验一致，区别在于Smirnov检验计算的是两个样本累积概率的最大差值，当然统计量分布和临界值表肯定也不一样。

    此方法假设总体的参数是已知的，但这在实际应用中是很难做到的。在这种情况下通常用样本均值和标准差代替总体均值和标准差，这就是所谓的Lilliefors正态性检验。

  • Lilliefors检验

    Lilliefors检验是基于Kolmogorov-Smirnov检验的正态检验。被用来测试H0不指定其正态分布，即没有指定期望值和方差。

    流程为采用样本数据估算总体平均值和总体方差，找到经验分布函数和正态分布累计分布函数之间最大区别，评估最大差异是否足够大到其统计的显著性，同需查表。

  • Shapiro–Wilk W检验

    K-S检验统计量用的是经验累积概率与目标理论累积概率之差的最大值，这有点像计算极差；AD检验则把所有的差平方后求和，有点像计算方差。当然计算方差的稳健性(Robustness)要好一些，这可能也是AD检验得到更多应用的原因。

    Shapiro–Wilk W检验，按照维基百科中，W检验类似是计算相关系数，这个数越接近1，则越有可能正态。
